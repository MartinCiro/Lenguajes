Motores de tablas

Tablas ISAM: Fue una tabla secuencial que avanza de a bloque 1-1.

Tablas MyISAM: Se generan dos archivos distintos, establecen la estructura de la tabla. Es la que se usa en el hoy

  • Estaticas: Tienen un tamaño definido y solo pueden almacenar un tipo de dato, todos los registros ocupan el mismo(maximo) espacio independientemente si es mayor o menos que el registro.
  • Dinamicas Intentan optimizar el tamaño, son mas lentas que las Estaticas debido a que asigna el tamaño a cada registro.
  • Comprimidas: Se compacta la informacion, no son muy usadas en el uso comun, se hace uso en servidores

Motor InnoDB Es el usado actualmente, lo asigna en automatico

Memory Era el motor HEAP, hace uso de la memoria RAM para guardar los datos, no se guarda en el disco duro, es optimo pero es volatil(Cuando se cierra/apaga toda la informacion de la memoria RAM se borra).

Merge: Agrupa tablas de tipo MyISAM siempre y cuando tengan los mismos tipos de datos, se usa para respaldos
