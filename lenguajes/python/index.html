Python

Es un lenguaje de tipado dinamico, es decir, no es necesario declarar el tipo de dato

Variables

a="5" -> 'a' es una variable tipo string
a=5 -> 'a' es una variable tipo int

Operaciones

Se maneja con el metodo PEMDAS (Parentish, Exponente, Multiplicacion, Division, Adicion, Abstraccion)

print 2 + 3 * 4 ** 2/ (5-1) -> 26.0
