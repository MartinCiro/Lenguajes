Python

Es un lenguaje de tipado dinamico, es decir, no es necesario declarar el tipo de dato

Todo parece imposible hasta que se hace. Nelson Mandela

Variables

a="5" -> 'a' es una variable tipo string
a=5 -> 'a' es una variable tipo int

Captura de datos por teclado

Esto se hace con input y todo será almacenado como texto, despues es necesario convertir el valor al tipo de dato necesitado(Ir a conversion de datos)

input("Mensaje a mostrar: ") -> 'Mensaje a mostrar' (Si el valor no es asignado a una variable será solo un mensaje en pantalla)
b = input("Mensaje a mostrar: ") -> 'Mensaje a mostrar' (El valor ingresado por teclado es asignado a una variable llamada b, accedemos al valor usando algún método de impresión o lo empleamos en una operación)

Impresion de mensajes

Usamos la palabra reservada print y si queremos que imprima el mensaje con un salto de linea Usamos println, para versiones como la 2 e inferiores debemos hacer uso de parentesis

print 2 + 3 * 4 ** 2/ (5-1) -> 26.0

Orden de operaciones

Se maneja con el método PEMDAS (Parentish, Exponente, Multiplicacion, Division, Adicion, Abstraccion)

print 2 + 3 * 4 ** 2/ (5-1) -> 26.0
